# SiteGen AI – Geração instantânea de sites para restaurantes

![Python](https://img.shields.io/badge/python-3.11-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-1.38-red)
![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4o--mini-green)

**Projeto criado em novembro/2025 especialmente para o Programa de Estágio GenAI do iFood**

## O que faz?
Em alguns segundos gera um site completo, bonito e responsivo (HTML + CSS) para qualquer restaurante, perfeito para parceiros de aplicativos como o iFood que ainda não têm presença digital.

### Funcionalidades
- Geração com GPT-4o-mini (rápido e barato)
- Interface 100% em português
- Download instantâneo do site
- Totalmente personalizável

## Como rodar a partir desse repositório

```
git clone https://github.com/EricC-B/simple-sitegen.git
cd sitegen-ifood
pip install -r requirements.txt
streamlit run app.py
```
Cole sua API Key do OpenAI e já pode começar a gerar as LPs.
